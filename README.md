# 星迹 StarryChat

一个基于 Node.js + WebSocket 的实时聊天室，支持多房间、私聊、文件发送、AI 助手、API 开放平台等功能。

## ✨ 功能特性

- 🏠 多房间聊天（大厅 + 自定义房间）
- 💬 私聊（双击成员列表）
- 🤖 内置 AI 助手（@ai 唤醒，免费 Pollinations API）
- 📎 文件/图片发送（带下载按钮）
- 🖼️ 视频通话（WebRTC）
- 🌙 深色/浅色主题切换
- 🔍 聊天记录搜索
- 😊 表情包面板
- ⏱️ 消息时间戳、撤回
- 🔔 新消息提示音
- 📱 手机端适配（侧边栏滑动）
- 🛡️ 管理员面板（踢人、广播）
- 📦 PWA 支持（可安装到桌面）
- 🔑 API 开放平台（开发者注册、密钥管理、调用限频）
