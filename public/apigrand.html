API 文档

POST /api/call

获取在线用户列表

fetch('/api/call', { method:'POST', headers:{'Content-Type':'application/json'}, body:JSON.stringify({apiKey:'YOUR_KEY'}) })

POST /api/send

发送系统消息

fetch('/api/send', { method:'POST', headers:{'Content-Type':'application/json'}, body:JSON.stringify({apiKey:'YOUR_KEY', message:'Hello'}) })

POST /api/history

获取聊天记录

fetch('/api/history', { method:'POST', headers:{'Content-Type':'application/json'}, body:JSON.stringify({apiKey:'YOUR_KEY', room:'大厅'}) })

POST /api/resetkey

重置密钥

fetch('/api/resetkey', { method:'POST', headers:{'Content-Type':'application/json'}, body:JSON.stringify({apiKey:'YOUR_KEY'}) })
